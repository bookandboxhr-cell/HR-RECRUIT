ในฟอร์มสมัคร ควรเพิ่มแนะนำ
  • + ช่องเงินเดือนที่คาดหวัง — คัดกรองเบื้องต้นได้ทันที ไม่ต้องโทรถามทีหลัง
  • + ช่อง "รู้จักจากช่องทางไหน" — วัดว่าประกาศช่องไหนคุ้มที่สุด (ลดงบโฆษณาที่ไม่ได้ผล)
  • + แนบเรซูเม่/รูปถ่าย — ดูข้อมูลครบก่อนเรียกสัมภาษณ์
  • + checkbox ยินยอม PDPA — จำเป็นตามกฎหมายเมื่อเก็บข้อมูลส่วนบุคคล
2
ในระบบติดตาม ควรเพิ่มลดขั้นตอน
  • + รหัสผู้สมัคร/ตำแหน่งอัตโนมัติ (เช่น REQ-2568-01) — อ้างอิงง่าย ไม่สับสนเวลามีชื่อซ้ำ
  • + ตัวนับระยะเวลาสรรหา (วันเปิดรับ → วันรับเข้า) — เห็นว่าตำแหน่งไหนหาคนยาก วางแผนล่วงหน้าได้
  • + สถานะ "สมัครใหม่" เพิ่มก่อน "รอสัมภาษณ์" — แยกคนที่ยังไม่ได้คัดกรองออกจากคนที่นัดแล้ว
  • ✓ บันทึกประวัติผู้สมัครเก่า (คลังผู้สมัคร) — เพิ่มแล้ว: คนที่เคยสมัครแต่ไม่ผ่าน เรียกกลับมาเมื่อมีตำแหน่งใหม่ที่เหมาะกว่าได้
